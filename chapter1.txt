Web 3.0 – 
It refers to the evolution of web utilization and interaction which includes altering the Web into a database, with the integration of DLT (Distributed Ledger Technology blockchain is an example) and that data can help to made Smart Contracts based on needs of individual. It enables the up-gradation of the back-end of the web, after a long time of focus on the front-end (Web 2.0 has mainly been about AJAX, tagging, and another front-end user-experience innovation). Web 3.0 is a term that is used to describe many evolutions of web usage and interaction among several paths. In this, data isn’t owned but instead shared but still be , where services show different views for the same web / the same data.
